# 如何让个人主页变成朋友那种效果

## 重要：你现在看的不是个人主页

- `https://github.com` → **首页动态流**（你截图里的样子，永远不会显示 Profile README）
- `https://github.com/xiaodong2771280890-sketch` → **个人主页**（朋友那种横幅会出现在这里）

朋友的页面是：https://github.com/wangkang1019

---

## 发布步骤（约 3 分钟）

### 1. 新建特殊仓库

1. 打开：https://github.com/new
2. **Repository name** 必须完全等于你的用户名：
   ```
   xiaodong2771280890-sketch
   ```
3. 选择 **Public**
4. **不要**勾选 Add a README
5. 点击 Create repository

### 2. 上传本文件夹内容

在 PowerShell 中执行：

```powershell
cd d:\Desktop\000\xiaodong2771280890-sketch
git init
git add .
git commit -m "Add premium GitHub profile README"
git branch -M main
git remote add origin https://github.com/xiaodong2771280890-sketch/xiaodong2771280890-sketch.git
git push -u origin main
```

如果提示登录，按浏览器提示完成 GitHub 授权即可。

### 3. 打开正确的页面

访问：

**https://github.com/xiaodong2771280890-sketch**

你会看到顶部大横幅 + 打字动画 + 技术栈 + 数据卡片。

### 4. 可选：让左侧资料更像朋友

打开：https://github.com/settings/profile

- **Name**：填显示名，例如 `Xiaodong`
- **Bio**：例如 `❤️ I love building systems! 🎓 Exploring AI & Urban Safety`
- **Email**：勾选公开邮箱（或在 README 里改成你的邮箱）
